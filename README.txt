Подготовка к запуску:
1. Для запуска приложения r_cfx_script необходимо установить 
язык программирования R и программу Rstudio.
ссылки для установки: 
R - https://cran.rstudio.com/
Rstudio - https://posit.co/download/rstudio-desktop/

2. Дирректория, из которой будет происходить запуск скрипта, 
должна содержать файл entry.txt и папки CFX_files и archive.

3. В дирректории CFX_files должны находиться только файлы с 
расширением .csv, полученные с приборов BioRad CFX.

4. Наличие .csv файлов, другого формата или пустых .csv файлов
ведет к некорректной работе программы и остановке с ошибкой.

5. Файлы других форматов обработаны не будут.

6. Скрипт процессирует только те файлы, которых нет в архиве.

7. После завершения копирования в гугл таблицу файлы переносятся
в архив.

8. Для работы скрипта необходим интернет.

9. Для записи в гугл таблицу нужны права редактирования для 
выбранной таблицы.

10. Запись в гугл таблицу производится от имени того, кто запустил
скрипт - нужна авторизация. 

-------------------------------------------------------------------
Запуск скрипта:
1. Открыть файл r_cfx_script.R в Rstudio (двойной клик левой кнопкой
мыши по файлу, либо 1 клик правой кнопкой мыши, долее выбрать "открыть 
с помощью - Rstudio")

2. Запуск скрипта производится при помощи одновременного нажатия
клавиш CTRL+Shift+Enter, либо CTRL+Alt+R, либо из меню - выбоать
Code -> Run Region -> Run All

3. Не интерактивный запуск (из командной строки) недоступен, 
поскольку необходимо пройти процедуру авторизации, которая 
производится через браузер.

4. Прохождение процедуры авторизации:
    * после запуска программы в консоли R (левый нижний угол Rstudio)
        появится сообщение типа:
        Is it OK to cache OAuth access credentials in the folder 
            ~/.cache/gargle between R sessions?
            1: Yes
            2: No
        
        Необходимо ввести 1 или 2 (если ввести 1, то больше данное 
        сообщение появляться не будет)
    
    * после выбора опции хранения данных авторизации откроется окно
    браузера на страница авторизации в гугл акаунт с сообщением:
        Choose an account
        to continue to Tidyverse API Packages
    * необходимо выбоать аккаунт с доступом к гугл таблице в которую 
    будет происходить запись
    * если потребуется, необходимо ввести пароль и код двойной
    аутентификации для выбранного аккаунта
    * далее откроется следующая страница с сообщением:
        Tidyverse API Packages wants 
        access to your Google Account
    * необходимо поставить галочки напротив опций
        * See, edit, create, and delete all of your 
            Google Drive files
        * See, edit, create, and delete all your 
            Google Sheets spreadsheets.
    * нажать "продолжить"

5. После прохождения авторизации выполнение скрипта будет
автоматически продолжено.

6. После успешного завершения выполнения программы в консоли R 
появится сообщение:

> [1] "Congratulation! Files successfully uploaded!"
